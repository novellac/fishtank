Project description: Why even make a fish tank?

It seemed fun!

Like a lot of frontend devs, my days consist mostly of making forms and lists accessible and aesthetically pleasing. I don't often get the chance to work with CSS animations or CSS art. This was a fun, calming way to learn some stuff and make something pretty. It was also fun to experiment with using CSS Grid as a complement/suplement for absolute positioning.

Cool, can I make a fish?

Yes! If you want to get into CSS art, or want to show off your skills, you're welcome to fork the GitHub repo and make a PR to contribute a fish.

Resources I used...

  • GitHub gist for lightening and darkening colours in CSS
  • YouTube video of CSS aquarium by NoviceCoder was helpful for learning about clippy site and waves.
  • Bennett Feely's clip paths generator was helpful for making clip paths with point and click.
  • CSS Tricks Clip Path almanac entry for more about what a clip path is, and how to read the values.
  • Different waves styles for finding some ideas about waves I liked.
  • Sara Soueidan's animating shapes article for knowing about transitions between complex polygons.
  • YouTube video making a CSS Fish for an animated fish using multiple divs
  • CSS Reset for taking away CodePen's (usually) helpful styling.
  • CSS Icons tutorial for bendy seaweed needs.
  • Alvaro Montoro's blog post on accessible CSS art for accessibility techniques.
  • S.O. post on animating a potato for making the fish swim in circles.
